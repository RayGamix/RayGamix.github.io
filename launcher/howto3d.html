To equip 3D items in RayGamix, follow these steps 🚀

  1. Open GitHub
  2. Create an account or sign in.
  3. Create a new repository.
  4. Make sure "Add a README file" is enabled when creating the repo.
  5. Upload your 3D model files (.glb, .fbx, .obj, etc.).
  6. Open the repository Settings.
  7. Go to Pages in the left sidebar.
  8. Under Branch, change None to main and save.
  9. Wait a few minutes for GitHub Pages to publish your files.
  10. Copy the final file link. Example:
    https://yourname.github.io/repository/model.glb
  11. Paste the link into the 3D item input in RayGamix and save.
  12. Adjust the X, Y, and Z values if needed.
