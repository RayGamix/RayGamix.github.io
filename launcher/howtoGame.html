To make a game in RayGamix, follow these steps 🚀

  1. Open GitHub
  2. Create an account or sign in.
  3. Create a new repository.
  4. Make sure "Add a README file" is enabled when creating the repo.
  5. Upload your game file (index.html and sometimes style.css and script.js) and your preview image file (filename.png or .jpg or whatever filetype you want)
  6. Open the repository Settings.
  7. Go to Pages in the left sidebar.
  8. Under Branch, change None to main and save.
  9. Wait a few minutes for GitHub Pages to publish your files.
  10. Copy the final file link. Example:
    https://yourname.github.io/repository/index.html
  11. Press the "Publish a Game" button in RayGamix, and paste the link in the URL input ( https://yourname.github.io/repository/index.html)
  12. Enter the Game's name in the Name input. ( Game name)
  13. Enter the Preview image URL in the Image input. ( https://yourname.github.io/repository/image.filetype)
That's it! You just made a game! Earn tokens when players play your game.
